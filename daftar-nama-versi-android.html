Daftar Nama Versi Android

Admin November 4, 2024

Daftar Nama Versi Android - Dalam dunia teknologi, Android telah menjadi salah satu sistem operasi mobile yang paling populer. Dengan rilisul baru setiap tahun, versi Android juga berbeda-beda dan terus dikembangkan untuk meningkatkan fitur dan kinerja. Pada artikel ini, akan kita bahas tentang daftar nama versi Android, mulai dari versi awal hingga versi terbaru.

Sejarah Awal Android

Sebelum kita membahas daftar nama versi Android, marilah kita ber nghari ke sejarah awal Android. Android pertama kali diproduksi oleh Android Inc. pada tahun 2003 dan dibeli oleh Google pada tahun 2005. Pada tahun 2007, Android versi 1.0 rilis dan diikuti oleh versi 1.5 dan 1.6. Namun, tidak semua versi Android ini dikenal dengan nama yang kita gunakan sekarang.

Daftar Nama Versi Android

Berikut adalah daftar nama versi Android yang telah dirilis sejak awal:

Versi Awal (2007-2008)

  • Android 1.0: API Level 1
  • Android 1.1: API Level 2
  • Android 1.5: API Level 3 (Cupcake)
  • Android 1.6: API Level 4 (Donut)

Versi Pertengahan (2009-2011)

  • Android 2.0-2.1: API Level 5-6 (Éclair)
  • Android 2.2-2.3: API Level 7-8 (Froyo)
  • Android 2.4: API Level 9 (Gingerbread)

Versi Baru (2011-2014)

  • Android 3.0-3.2: API Level 11-13 (Honeycomb)
  • Android 4.0-4.3: API Level 14-16 (Ice Cream Sandwich, Jelly Bean)
  • Android 4.4: API Level 19 (KitKat)

Versi Terbaru (2014-sekarang)

  • Android 5.0-5.1: API Level 21-22 (Lollipop)
  • Android 6.0-6.0.1: API Level 23 (Marshmallow)
  • Android 7.0-7.1: API Level 24-25 (Nougat)
  • Android 8.0-8.1: API Level 26-27 (Oreo)
  • Android 9.0: API Level 28 (Pie)
  • Android 10: API Level 29 (Quebec)
  • Android 11: API Level 30 (Red Velvet Cake)
  • Android 12: API Level 31 (Snowcone)
  • Android 13: API Level 33 (Tiramisu)
  • Android 14: API Level 36 (Upside Down Cake)

Pengembangan dan Perbaikan

Kemajuan perkembangan Android sangat cepat dan terus berkembang. Pada tahun 2009, Android 2.0 Éclair dirilis dengan fitur baru seperti gestur multitouch dan browser baru. Pada tahun 2011, Android 3.0 Honeycomb dirilis dengan fitur tablet bahkan tanpa keyboard. Pada tahun 2013, Android 4.4 KitKat dirilis dengan fitur widget baru daninya perbaikan kinerja.

Fitur Baru dan Perbaikan

Berikut beberapa fitur baru dan perbaikan yang terkait dengan versi Android:

  • Gestur multitouch (Android 2.0 Éclair)
  • Browser baru (Android 2.0 Éclair)
  • Fitur tablet (Android 3.0 Honeycomb)
  • Widget baru (Android 4.4 KitKat)
  • Perbaikan kinerja (Android 4.4 KitKat)
  • Material Design (Android 5.0 Lollipop)
  • Fitur Lockscreen (Android 5.0 Lollipop)
  • Doze Mode (Android 6.0 Marshmallow)
  • Vulkan API (Android 7.0 Nougat)
  • Google Play Protect (Android 7.0 Nougat)
  • Autofill APIs (Android 8.0 Oreo)
  • Picture-in-Picture Mode (Android 8.0 Oreo)
  • Android Go (Android 8.1 Oreo)

Kesimpulan

Dalam kesimpulan, daftar nama versi Android telah berkembang pesat sejak awal dan terus dikembangkan untuk meningkatkan fitur dan kinerja. Berikut adalah beberapa fitur baru dan perbaikan yang terkait dengan versi Android. Untuk pengembang aplikasi, pengetahuan tentang daftar nama versi Android sangat penting untuk memastikan aplikasi yang Anda buat dapat berjalan dengan baik pada berbagai versi Android.
